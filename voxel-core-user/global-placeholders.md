# Global Placeholders

## Formatting Numbers

Let's take input of `123456789`

| Variable          | Output                         |
|-------------------|--------------------------------|
| `variable`        | `123456789`                    |
| `variable_commas` | `123,456,789`                  |
| `variable_sl`     | `123.457 million`              |
| `variable_ll`     | `123 million 456 thousand 789` |
| `variable_ls`     | `123 M 456 K 789`              |
| `variable_ss`     | `123.457M`                     |
